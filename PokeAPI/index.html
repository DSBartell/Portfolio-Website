How to Use this Web App

This web app uses the fan made Pokemon API to grab data about various Pokemon and display their base statistics with the Google Charts API. If you are not familiar with Pokemon, try entering a number from 1 - 803 (the total number of pokemon). A few of my favorites are Jirachi, Mew, and of course Pikachu.

If for some reason this web app is not working for you, it's likely because there is a CORS Error. This is because I haven't made a proxy server or functional equivalent yet. To bypass this issue, I reccomend that you download a CORS Chrome Extension here. ONLY ENABLE IT WHEN YOU'RE EITHER DEVELOPING WEB APPS OR MAKING CORS REQUESTS, it will mess up some websites if you dont. Feel free to modify this program and push it to Github. I'd love to see what you make! This project is not yet completed, so stick around for more updates!
